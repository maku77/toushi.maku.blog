原資産からの派生商品（デリバティブ）は、大きく分けると次の 3 つに分類できます。

- {{< label "スワップ" >}}（金融機関同士の相対取引）
  - 金利スワップ（店頭デリバティブ取引の 70% を占める）
  - クレジット・デフォルト・スワップ (CDS)
  - バリアンス・スワップ
- {{< label "フューチャー" >}}（先物）
  - 商品先物
  - 金融先物
    - 通貨先物（為替先物）
    - 金利先物
    - 株価指数先物
- {{< label "オプション" >}}

{{% private %}}
- 為替予約
- 通貨オプション
- 天候デリバティブ
- 差金決済取引 (CFD)
{{% /private %}}
